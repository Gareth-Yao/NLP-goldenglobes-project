To install requirements:
- run: pip freeze > requirements.txt
- run: python install -r requirements.txt

Running gg_api.py
- run preceremony: python gg_api.py pre_ceremony year
- run main: python gg_api.py year

Please make sure the tweet json file with the year you are trying to run is in the directory. For example, put gg2018.json in this directory.
